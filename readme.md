# Програма для рассчета расхода топлива для разных типов уведомления водителя о расходе
После запуска программы введите значение 1 или 2 или 3, в зависимости от того, какой тип значения расхода топлива используется на вашем автомобиле.
После ввода значения, программа выдаст два других различных значения с появснениями в какой части мира такой способ записи расхода топлива используется.
